## 專案設定與初始化：
- 使用 Next.js 建立專案架構。
- 配置基本的路由和頁面。
- 設計對應的 RWD（響應式網頁設計）。

## 串接 YouBike Open API：
- 實現 API 串接，抓取台北市 YouBike 站點即時數據。
- 為其它縣市配置假數據或預留空白區域。

## 縣市搜尋功能：
- 開發一個搜尋輸入框，用於篩選縣市。
- 搜尋結果用於控制縣市清單的展示/收合。
- 添加一鍵清除搜尋內容的功能。

## 行政區篩選功能：
- 渲染所有台北市行政區的 checkbox。
- 實現全選與個別選擇的邏輯，勾選或取消後，更新其餘 checkbox 的狀態。

## Header 組件：
- 開發一個能夠在點擊後切換路由的 Header 組件。
- 路由的規劃應符合網站結構，不需在點擊後改變頁面內容。
